installation¶

DChars est un module pour des projets utilisant Python3. Récupérez la dernière version de DChars, placez-vous dans le répertoire de votre projet et tapez tar -xvzf dchars.tar.gz. Un sous-répertoire dchars/ est alors créé avec tout le code nécessaire.
