DChars¶

DChars (Detailed Characters) est un module pour Python3 permettant de modifier facilement les signes diacritiques de caractères unicodes complexes et d’en obtenir une translittération. DChars est un module libre (GPL3) et gratuit. Les langues actuellement reconnues sont le grec ancien, le latin, l’hébreu (biblique), le sanskrit (écrit en devanāgarī) et le tibétain.

Téléchargez la dernière version :

► dchars_v0.3.9.tar.gz

Si vous êtes intéressé(e), jetez un œil au répertoire de développement (► dev) ou au diagramme UML de l’ensemble des classes (► UML).
Dans tous les cas, n’hésitez pas à écrire à l’auteur : suizokukan, <suizokukan _A.T_ orange•fr>
